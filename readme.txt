knapsack.py is a solution for the multiple 0-1 knapsack problem. To run, ensure the file named shopping.txt is included in the same directory.
Next, type $python3 knapsack.py
The results will be outputted to a file named results.txt.

last-to-start.py is a greedy solution to the activity solution problem. Instead of the normal greedy solution, it sorts the start times in decreasing order, with the latest starting time first.
To run, ensure a file named act.txt is in the same directory. 
Next, type $python3 last-to-start.py
The solution will be outputted to the console.
